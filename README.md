- Quản lý đánh giá và bình luận

- Tích hợp chatbot(Tidio)

## 🛠 Cách cài đặt và chạy dự án  
**1. Clone repo về máy:**  
```bash
git clone https://github.com/TranChinh2901/ecommerce_mern_stack.git
cd ecommerce_mern_stack
```
**2. cd file cần thiết:**
```bash
cd Frontend     -> npm install
cd Backend      -> npm install
```
**3. Chạy dự án:**
```bash
Phía  Frontend:    -> npm run dev
Phía  Backend:     -> npm start
```
